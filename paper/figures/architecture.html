4 Evaluation
Deterministic
Shell command → parse numeric score. Retry + flake detection.
Stochastic
N samples × K binary criteria. Bradley-Terry scoring, position-bias cancellation.
↓
5 Statistical Decision
Wilcoxon signed-rank test (p < α). Holm–Bonferroni correction for sequential multiplicity. Effect-size fallback for small samples (n < 6).
